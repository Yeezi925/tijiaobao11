# 体教宝 - 现代化体育成绩系统设计方案

## 选定设计风格：现代极简主义 + 运动活力

### 设计哲学
**Modern Minimalism with Athletic Energy** - 融合专业教育工具的严谨性与运动领域的活力感。通过清晰的信息架构、流畅的交互和精准的数据可视化，为体育教师和学生提供高效的成绩管理体验。

---

## 核心设计原则

### 1. **信息优先级明确**
- 学生成绩卡片采用分层设计：总分最突出，贡献分项次之，详细数据可展开
- 导航和操作按钮清晰分区，避免信息过载

### 2. **数据可视化直观**
- 采用柔和的渐变色彩区分不同项目和性别
- 使用进度条、环形图表展示成绩分布
- 表格设计宽敞，行高充足，易于扫描

### 3. **交互反馈即时**
- 按钮悬停时有微妙的阴影和颜色变化
- 文件上传、数据导入有加载动画和成功提示
- 表格行选中时高亮显示

### 4. **响应式设计**
- 移动端：单列布局，卡片式展示
- 平板端：两列网格，表格横向滚动
- 桌面端：充分利用宽度，多窗格布局

---

## 色彩哲学

### 主色系：运动蓝 + 活力绿
- **主色（蓝）**：`#1976D2` - 专业、信任、稳定感
- **辅助色（绿）**：`#4CAF50` - 成功、进步、活力
- **警告色（橙）**：`#FF9800` - 需要关注的项目
- **中性色**：灰度系列用于文本和背景

### 背景与文本
- **亮色模式**（默认）：
  - 背景：`#F8F9FA` - 柔和灰白
  - 卡片：`#FFFFFF` - 纯白
  - 文本：`#1A1A1A` - 深灰黑
  - 次文本：`#666666` - 中灰

## 布局范式

### 页面结构
```
┌─────────────────────────────────────┐
│     Header (Logo + 导航 + 用户菜单)   │
├─────────────────────────────────────┤
│  Sidebar (可选)  │  Main Content     │
│  - 导入          │  - 学生列表       │
│  - 查询          │  - 统计分析       │
│  - 分析          │  - AI 建议        │
└─────────────────────────────────────┘
```

### 关键区域
1. **顶部导航**：Logo、项目名称、快速操作按钮
2. **侧边栏**（桌面）：功能导航，可收起
3. **主内容区**：根据选中功能动态显示
4. **底部**：版权、帮助链接

## 签名元素

### 1. **成绩卡片**
- 圆角矩形（radius: 12px）
- 顶部彩色条带（根据成绩等级变色）
- 投影：`0 2px 8px rgba(0,0,0,0.08)`
- 悬停时投影加深，卡片微微上浮

### 2. **进度条与环形图**
- 使用渐变色（蓝→绿）表示成绩进度
- 环形图用于展示四个维度的贡献分（长跑、球类、选考1、选考2）
- 动画：从 0 到目标值的平滑过渡（500ms）

### 3. **数据表格**
- 表头：浅蓝背景 + 深灰文本
- 行间距：1.2 倍行高，易于阅读
- 交替行背景：`#F5F5F5` 和 `#FFFFFF`
- 可交互行：悬停时背景变浅蓝，鼠标指针变手形

---

## 交互哲学

### 文件上传
- 拖拽区域：虚线边框 + 浅蓝背景
- 拖入时：边框变实线，背景加深
- 上传中：加载动画 + 进度百分比
- 完成：绿色勾号 + 成功提示

### 表格操作
- 单击行：整行高亮 + 侧边出现操作按钮（查看详情、生成建议）
- 多选：复选框 + 批量操作工具栏
- 排序：点击列头排序，显示排序指示器（↑/↓）

### 模态框与抽屉
- 打开：淡入 + 内容从下方滑入
- 关闭：淡出 + 内容向下滑出
- 背景：半透明黑 (rgba(0,0,0,0.5))

---

## 动画指南

### 页面加载
- 卡片列表：瀑布式进入（每张卡片延迟 50ms）
- 表格行：从左向右淡入

### 交互反馈
- 按钮点击：按下 → 释放（缩放 0.98 → 1.0）
- 悬停效果：背景色 + 投影变化（200ms）
- 数据更新：数字滚动动画（从旧值到新值）

### 过渡
- 页面切换：淡出 → 淡入（300ms）
- 侧边栏展开/收起：宽度变化（250ms）
- 模态框：缩放 + 淡入（300ms）

## 排版系统

### 字体配对
- **标题字体**：`Poppins` (Google Fonts) - 现代、友好、略带活力
  - H1: 32px, 700 weight, 行高 1.2
  - H2: 24px, 600 weight, 行高 1.3
  - H3: 20px, 600 weight, 行高 1.4

- **正文字体**：`Inter` (Google Fonts) - 清晰、中性、高易读性
  - 正文: 14px, 400 weight, 行高 1.6
  - 小文本: 12px, 400 weight, 行高 1.5
  - 强调: 14px, 600 weight

### 层级规则
1. 页面标题 (H1) - 最大、最重
2. 区域标题 (H2) - 中等、加粗
3. 卡片标题 (H3) - 较小、加粗
4. 正文文本 - 标准大小、常规
5. 辅助文本 - 较小、灰色

### 间距
- 段落间距：1.5rem
- 卡片内间距：1.5rem
- 列表项间距：0.75rem
- 按钮内间距：0.75rem 1.25rem
